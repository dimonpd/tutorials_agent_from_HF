# Hub Tutorials

AI agent tutorial repository with scripts, notebooks, and presentation materials.

## Skills

- `/slides` - Generate Reveal.js presentation slides from tutorial scripts or topics

## Project Structure

- `src/` - Tutorial source code projects
- `notebooks/` - Jupyter notebooks
- `slides/` - Generated presentation HTML files
- `TUTORIAL_IDEAS.md` - List of potential tutorial topics

## Conventions

- Tutorial scripts use SCRIPT.md format
- Slides output to `slides/{project-name}/`
- Use OpenAI Agents SDK patterns for multi-agent tutorials
